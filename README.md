## 🎮 How to Use

### Main Menu Options
```
[1-11]  Open a phase
[r]     View your profile & rank
[s]     Search for topics/tools
[b]     Manage bookmarks
[p]     View progress report
[a]     View badges earned
[c]     CTF Challenge mode
[t]     Change theme (dark/light)
[0]     Exit
```

### Topic Menu Options
```
[1]     View & open tools
[2]     Watch tutorial (if available)
[3]     View tool tutorials (if available)
[b]     Add/remove bookmark
[d]     Mark topic as done/undone
[0]     Go back
```

### Phase Menu Options
```
[v]     Watch phase playlist (if available)
[topic] Open topic
[d]     Mark topic done/undone from topic menu
[0]     Go back
```

## 🐛 Troubleshooting

**Issue: Files not found**
- Make sure `main.py` and `data.json` are in same folder

**Issue: Links not opening**
- Check your internet connection
- Some systems may require manual browser opening

**Issue: Colors not showing properly**
- Use a modern terminal (not very old CMD)
- Try theme option `[t]` to switch themes

**Issue: Tools not installing on Linux**
- Make sure you have `sudo` permissions
- Some tools might require manual installation
